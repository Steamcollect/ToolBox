Property Drawer :
	-ShowIf : show variable with condition
	-HideIf : hide variable with condition
	-Button : draw button on inspector
	-Tab : Separe le component en plusieurs onglets
	-Foldout : Foldout ouvrable et fermable
	-SceneName : dropdown of every scene in build settings
	-ReadOnly : show variable on inspector but cant be modified
	-Dropdown : show dropdown base array or reference
	-Watch : display the variable and its value on screen
	-Inline : Dessine une class ou struct sur le moins de ligne possible (sans foldout)

Class :
	-InterfaceReference : show on inspector the interface as object reference

Utils :
	-IEnumerableUtils
	-CoroutineUtils
	-RotationUtils

Wrappers :
	-Runtime Scriptable Object
	-Runtime Scriptable Event
